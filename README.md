Chalo Bhaiya - Ride Sharing for MU Students
Chalo Bhaiya is a community-driven ride-sharing web application designed specifically for the students of Marwadi University. The platform aims to simplify the daily commute by connecting students who need a ride with those traveling in the same direction, helping everyone save money and reduce the campus's carbon footprint.

✨Key Features:
Post a Ride: Students can easily post their intended departure time and destination (e.g., Rajkot Bus Stand, Crystal Mall, or University Campus).
Find a Ride: A real-time list of available rides that students can join.
Cost Calculator: An integrated tool to estimate and split auto-rickshaw fares among passengers.
MU Authentication: A secure login system to ensure the platform is used exclusively by verified Marwadi University students.
Responsive Design: Fully optimized for both desktop and mobile use, ensuring students can find rides on the go
